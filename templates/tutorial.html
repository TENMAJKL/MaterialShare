Col-system

Col systém je skupina tříd, které drží šířky využívané na webu, dělí se na 3 základní:

  • col-1; přes 3/3 stránky
  • col-2; přes 2/3 stránky
  • col-3; přes 1/3 stránky
  • col-half; přes 1/2 stránky
  • col-quarter; přes 1/4 stránky

V žádném případě nepoužívejte konstatní šířky a už vůbec ne v px!

Column, row

Row je třída, ve které jsou na PC všechny objekty vedle sebe, zatímco na mobilu jsou pod sebou. Pokud ale chceme objekty pod sebe i na pc, použijeme třídu column.

Mobile-row

Občas se stane, že na mobilu potřebujete věci vedle sebe, od toho je tu třída mobile-row, v ní jsou věci vedle sebe i na mobilu. v mobile-row kromě col- používáme i m-col- tam dáváme stejnou hodnotu jako u col-

row-

Pro row existují tagy row-center a row-space, center vycentruje a space dá objekty co nejdál od sebe. Platí pro row a mobile row.

Text

Na webu se používá font Poppins. Pro text se používají tyto třídy:

  • text-center; Text veprostřed
  • text-right; Text vpravo
  • text-left; Text vlevo

Grid

Grid systém, se používá, pokud chceme určitý počet věcí vedle sebe a pokud jich máme víc, tak pod sebe. Jediné co stačí, tak je dát třídu container na div, kde máme itemy a pro lepší vizi na mobilu dát na každý item m-cod-item:

  • container-2; řadí po 2 vedle sebe
  • container-3; řadí po 3 vedle sebe
  • container-4; řadí po 4 vedle sebe

Flask

Úvod do Flasku

Flask je webový mikroframework pro python, na webu se používá pro backend a zde jsou informace co dělat jak dělat.

Routování

Pokud chceme udělat novou stránku používáme routování, to je způsob vytváření cest v adrese; Zde příklad jak vytvářet stránky pro testování

@app.route("/name")
def name():
    return render_template("jmeno_html_souboru")

Blueprinty

Blueprint je soubor, který obshuje routy, bluepritny používáme proto, aby byl kod roztříděný.

Proměnné, ify, cykly,...

Flask nám umožnujě dynamické webové stránky, proto můžeme do html teplatů vkládat netradiční věci

Proměnné

Proměnné vkládáme pomocí { {jmeno} }

Cykly

Pokud chceme něco cyklit (materiál), použijeme cyklus: { % for % } { % endfor % }

If

Pokud chceme použít if, máme tyto syntaxe: { % if % } { % else % } { % elif % } { % endif % }

Nepoužíváme mezery mezi {}

Formuláře

Formuláře fungují stejně, akorát na začátek dáme vždy toto:

<input type="hidden" name="csrf_token" value="{ { csrf_token() } }"/&tg

Opět mezi {} nepoužíváme mezery.

Jedná se o ochranu proti CSRF útokům.
